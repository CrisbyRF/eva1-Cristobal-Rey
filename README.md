--HUERTO FELIZ - LANDING PAGE--
Huerto Feliz es un proyecto de sitio web para servicios de jardinería, diseñado para ofrecer una experiencia visual atractiva y una navegación fluida.

Características:

* Diseño Responsive: Adaptado para dispositivos móviles, tablets y escritorio.
* Carrusel Dinámico: Exhibición visual de los servicios principales.
* Formulario de Contacto: Listos para capturar consultas de clientes potenciales.
* Secciones informativas: Historia, Empresa y Servicios detallados.

Tecnologías utilizadas:

* HTML5 - Estructura del sitio.
* CSS3 - Estilos personalizados y diseño visual.
* Bootstrap 5 - Framework para el layout y componentes responsivos.

Instalación y Uso

* Clona el repositorio: git clone https://github.com/CrisbyRF/eva1-Cristobal-Rey
* Abre el archivo index.html en tu navegador (para la evaluación: Google Chrome).
